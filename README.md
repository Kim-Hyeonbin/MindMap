# 🧠MindMap
🔗 https://kim-hyeonbin.github.io/MindMap/  
오픈소스 기반 심리 검사를 제공하는 웹페이지를 만드는 프로젝트입니다.
사용자의 성격적 특성, 사고 경향, 감정 상태 등을 간단한 검사와 시각화된 분석을 통해 살펴볼 수 있도록 설계되었습니다.

## ✅목표
- 사용자가 자신의 성격적 특성과 인지적 능력을 간단히 탐색할 수 있는 자기 이해 플랫폼 제작
- 신뢰할 수 있는 공개 심리 검사 데이터셋을 기반으로 한 간단한 분석 결과 제시

## 📋제공 서비스 및 페이지 요약
- 홈 화면에서 검사 선택 → 검사 설명 → 문항 응답 → 결과 분석 및 요약 출력의 4단계로 페이지 흐름 구성 

- 각 검사는 5~10분 정도의 짧은 시간으로 완료할 수 있는 분량으로
구성되며, 결과 화면에서는 각 요인 별 점수와 결과 요약을 제공 

## 🧐 제공 검사
1) IPIP Big Five 성격 유형 검사
- 국제적으로 널리 사용되는 Big Five 성격 모델 기반
- 외향성, 호감성, 성실성, 정서적 안정성, 개방성 5요인 분석

2) CRT(인지 반추 검사, Cognitive Reflection Test)
- 직관적 사고 vs 숙고적 사고 경향을 간단한 문제를 통해 측정
- 오답 유형(직관적 오답 / 일반 오답)에 따라 사고 성향 분류
- 글로벌 통계 기반 정답률 비교 제공

3) DASS 정서 상태 검사 (Depression–Anxiety–Stress Scales)
- 우울(D), 불안(A), 스트레스(S) 3요인 평가
- 정규분포 기반 백분위 계산 및 수준(level) 분류

## 🛠 기술 구성
- HTML/CSS/JS 기반 순수 프런트엔드 프로젝트
- Test 데이터 및 통계는 JSON 형태로 로드하여 처리
- Session Storage를 이용한 응답 전송 및 데이터 저장
- Chart.js 기반 시각화(일부 검사)
- GitHub Pages를 통한 정적 배포
