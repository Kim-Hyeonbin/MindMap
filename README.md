# 🧠MindMap
🔗 https://kim-hyeonbin.github.io/MindMap/  
오픈소스 기반 심리 검사를 제공하는 웹페이지를 만드는 프로젝트입니다.

## ✅목표
사용자가 자신의 성격적 특성과 인지적 능력을 간단히 탐색할 수 있는 자기 이해 플랫폼 제작

## 📋제공 서비스 및 페이지 요약
- 오픈소스 형태로 제공되는 성격 유형 검사인 IPIP 기반 Big five 성격 유형 검사와,마찬가지로 오픈소스인 ICAR 지능 검사 서비스를 제공

- 홈 화면에서 검사 선택 → 문항 응답 → 결과 분석 및 요약 출력의 3단계로 페이지 구성 예정

- 각 검사는 5~10분 정도의 짧은 시간으로 완료할 수 있는 분량으로
구성할 계획이며, 결과 화면에서는 각 요인 별 점수와 결과 요약을 제공 

- 프로젝트 진행 상황을 보고 OEJTS 성격 유형 검사 등 다른 심리 검사도 추가 고려
